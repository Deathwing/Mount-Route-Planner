# Mount Route Planner Data

**Mount Route Planner Data** is an optional standalone dependency package for Mount Route Planner.

It contains the generated mount source and route-step data consumed by the main addon, split into its own package so the runtime addon and its datasets can be distributed independently when needed.

### What This Package Contains

* Generated mount source data
* Generated route-step data
* Flavor-specific datasets for Retail, Mists Classic, Cataclysm Classic, Wrath Classic, Burning Crusade Classic, and Classic Era

### Who Should Install This

* Install this package only if you are using the standalone CurseForge package layout for Mount Route Planner.
* If your Mount Route Planner installation already includes embedded data, this package is optional and should not be installed separately.

### What This Package Does Not Do

* No UI
* No slash commands
* No minimap button
* No standalone functionality without the main Mount Route Planner addon

### Installation

For standalone dependency installs only:

1. Install **Mount Route Planner**.
2. Install **Mount Route Planner Data**.
3. Start the game and enable both addons.

Keeping both packages on matching versions is recommended.
